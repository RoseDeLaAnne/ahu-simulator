# Tooling

This directory contains scripts that support development and migration but are not required for runtime imports.

Subfolders:
- `migration/` for safe restructuring and cleanup helpers
- `scene/` for 3D generation scripts
- `commands/windows/` for click-to-run Windows launchers moved out of repository root
- `bin/` for auxiliary local executables used by tooling (for example cloudflared)
- `packaging/` for build and release helpers if moved from deploy scripts
