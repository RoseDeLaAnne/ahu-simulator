# data

Здесь хранятся:

- синтетические входные данные;
- шаблоны сценариев;
- экспорт результатов;
- вспомогательные таблицы для валидации.

Текущие подкаталоги и файлы:

- `scenarios/presets.json` — пресеты демонстрационных режимов;
- `validation/reference_points.json` — контрольные точки MVP для regression-проверок модели.
- `validation/reference_basis.json` — реестр внешних методических оснований и трассировки шагов/контрольных точек.
- `validation/validation_agreement.json` — согласованные control points, step tolerances и границы применимости фазы качества.
- `visualization/browser_capability_profile.json` — verified browser/demo-PC envelope для future optional WebGL-режима.
- `visualization/scene3d.json` — scene bindings и metadata для будущего 3D-viewer.
