# Backlog улучшений

## Ближайшие улучшения после MVP

- Отдельная вкладка «Сценарии».
- Сохранение пользовательских наборов параметров.
- Расширение `Export Pack` сравнительными выгрузками по нескольким прогонам.
- Шаблоны PDF-отчёта для приложений к ВКР и печатных материалов.
- Фильтрация и поиск по `Event Log`.
- Усложнение 2D-мнемосхемы после готового базового SVG-слоя.
- Полноценный 3D-viewer с камерой, сценой и детализированной геометрией.
- Реальный pipeline 3D-ассетов: `GLB/GLTF`, текстуры, naming convention узлов и регламент обновления сцены.
- Расширенный `scene3d.json` с camera presets, интерактивными узлами, flow-rules и metadata для fallback/performance.
- Optional 3D-mode switch в dashboard с безопасным возвратом к 2D при ограничениях WebGL.
- Hover/click-взаимодействие с узлами 3D-сцены и карточкой выбранного элемента.

## Инженерные улучшения

- Добавление динамики по времени.
- Упрощенная реализация ПИД-регулятора.
- Калибровка модели по контрольным точкам.
- Расширение `auto/manual` до отдельного ПИД-контура и набора управляющих команд.
- Расширение набора аварийных и предаварийных условий.
- Детализация 3D-представления по узлам установки и воздуховодам.
- Оптимизация 3D-сцены под verified browser/demo-PC envelope: budget по полигонам, texture-size, device pixel ratio и cleanup GPU-ресурсов.

## Интеграционные улучшения

- Теговая модель сигналов.
- Подготовка адаптера под внешнюю SCADA/OPC-интеграцию.
- Отдельный backend-mode без UI для будущих интеграций.
- Фильтрация, поиск и быстрые сравнения по локальному архиву прогонов.
- Экспорт scene metadata и signal bindings в формат, пригодный для внешнего 3D/SCADA-viewer без переписывания расчётного слоя.

## Академические улучшения

- Таблица сравнения нескольких вариантов модели.
- Раздел «чувствительность модели к параметрам».
- Отдельные графики для пояснительной записки.
- Автоматическая генерация приложений с таблицами режимов.

## Улучшения, которые пока не нужно делать

- Реальную интеграцию с оборудованием.
- Production-grade мониторинг и аутентификацию.
- Микросервисную декомпозицию.
- Переусложнение интерфейса раньше подтверждения модели.
- Полноценную 3D-визуализацию раньше готовности 2D-модели и visual bindings.

## Перенесено в активный план (2026-04-18)

- Расширение симуляции по времени: `Старт/Пауза/Сброс`, шаги времени.
- Отчетность по сценарию: `PDF/CSV`.
- Сравнение прогонов `до/после`.
- Цветные статусы `Норма/Риск/Авария`.
- Пресеты `Зима/Лето/Пик нагрузки`.

## Что остается в backlog после переноса

- Углубленная калибровка модели по полевым данным конкретной установки.
- Полная OPC/SCADA-интеграция с внешним теговым пространством.
- Отдельный режим backend-only для интеграционного контура предприятия.
- Расширенный набор инженерных метрик и sensitivity-анализ для ВКР.
