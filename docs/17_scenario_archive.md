# Scenario Archive

Дата актуализации: 2026-04-04.

## Где это доступно

- В dashboard добавлен отдельный блок `Scenario Archive`.
- Он доступен по якорю `#scenario-archive`.
- API-снимок архива доступен через `GET /archive/scenarios`.
- Сохранение текущего прогона через API доступно через `POST /archive/scenarios`.

## Что делает блок

- сохраняет текущий прогон в JSON без ручного копирования данных из интерфейса;
- различает готовый preset-сценарий и пользовательский набор параметров;
- показывает последние сохранённые записи прямо в dashboard;
- складывает файлы в `artifacts/scenario-archive/<дата>/`.

## Формат сохранения

- один JSON-файл соответствует одному сохранённому прогону;
- внутри хранится полный `SimulationResult` и метаданные источника;
- имя файла имеет вид `pvu-run-YYYYMMDD-HHMMSS.json`;
- если в ту же секунду уже есть запись, добавляется суффикс `-2`, `-3` и так далее.

## Зачем это добавлено

- чтобы закрыть практический P2-блок `архив сценариев` не в виде идеи, а как рабочий инструмент;
- чтобы фиксировать показанные на защите режимы и потом быстро к ним возвращаться;
- чтобы `Demo Bundle` мог захватывать не только код и документы, но и структуру локального архива.
