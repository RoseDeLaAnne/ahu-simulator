# Screenshots Artifact Area

Purpose:
- Store temporary and review screenshots generated during manual checks, local experiments, and migration runs.

Recommended structure:
- `manual/YYYY-MM-DD/`
- `playwright/YYYY-MM-DD/`
- `mobile/YYYY-MM-DD/`

Rules:
- Do not import these files from runtime code.
- Keep only evidence that is still useful for validation or release notes.
