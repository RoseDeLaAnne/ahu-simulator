# Demo Packages

Сюда складываются собранные offline/demo bundle артефакты.

## Структура

```text
artifacts/demo-packages/
  README.md
  YYYY-MM-DD/
    pvu-demo-package-YYYYMMDD-HHMMSS.zip
    pvu-demo-package-YYYYMMDD-HHMMSS.manifest.json
```

## Правила

- Каждый запуск сборки создаёт новый zip и соседний `manifest.json`.
- Папка с датой отражает день фактической сборки bundle.
- Внутри архива сохраняется исходная относительная структура проекта.
- При наличии локальных `CSV/XLSX/PDF` export-наборов bundle захватывает и их структуру, чтобы переносимые отчёты не терялись между показами.
- При наличии локального `Scenario Archive` bundle захватывает и его структуру, чтобы снимки прогонов не терялись между показами.
- Bundle предназначен для офлайн-показа, передачи проекта на следующий этап и быстрой повторной проверки состава поставки.
