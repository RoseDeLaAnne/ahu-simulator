# Event Log

Сюда складываются JSON-записи журнала событий проекта.

## Структура

```text
artifacts/event-log/
  README.md
  YYYY-MM-DD/
    pvu-event-YYYYMMDD-HHMMSS.json
    pvu-event-YYYYMMDD-HHMMSS-2.json
```

## Правила

- Каждый значимый расчётный переход, смена режима управления, экспорт и сохранение в архив пишутся отдельной JSON-записью.
- Папка с датой отражает день фактической фиксации события.
- Внутри записи сохраняются категория события, severity, источник, краткое summary и детали перехода.
