# Scenario Archive

Сюда сохраняются локальные JSON-снимки прогонов и сценариев.

## Структура

```text
artifacts/scenario-archive/
  README.md
  YYYY-MM-DD/
    pvu-run-YYYYMMDD-HHMMSS.json
    pvu-run-YYYYMMDD-HHMMSS-2.json
```

## Правила

- Каждый снимок сохраняется отдельным JSON-файлом.
- Внутри файла фиксируется полный `SimulationResult` и метаданные источника.
- Папка с датой отражает день фактического сохранения прогона.
- При повторном сохранении в ту же секунду к имени файла добавляется числовой суффикс.
